What is DownloadsManarr?

A media file organizer for your downloads folder. Select files or folders, configure where they should go, and move or copy them to the right destination — with optional renaming, subfolder wrapping, and season folder sorting.

Works entirely in your browser. No cloud. Your files stay on your machine.

How it Works

  1. 1Configure — Add source folders (where downloads land) and destination folders (where organized media goes) in ⚙ Settings → Library.
  2. 2Browse — Your source folders appear in the file list. Click any folder name to navigate inside it.
  3. 3Select — Check the files or folders you want to act on. The action bar appears above the list.
  4. 4Configure Action — Choose Move or Copy, pick a destination, and set options like rename or season folder.
  5. 5Queue or Run — Click Add to Queue to stage it for batch processing, or Run Now to execute immediately.

Key Concepts

📂
Sources & Destinations

Sources are folders DownloadsManarr scans (your Downloads folder). Destinations are where organized media lands — one per media type (TV, Movie, Music, Games…).

🏷️
Auto Type Detection

Filenames and folder names are automatically matched to media types using pattern recognition. You can override any detection using the type dropdown in the row.

📋
Queue vs Run Now

The queue lets you stage multiple different actions and run them all at once. Run Now skips the queue and executes a single action immediately.

Navigating Files

📁
Folder Navigation

Click a folder name to drill into it. A breadcrumb bar appears at the top — click any segment to jump back to that level. The .. row at the top of the list goes up one level. The ↻ refresh button always returns to the source root.

🔒
Locked Folders

A 🔒 badge means there is a queued action on or inside that folder. You can still navigate inside it, but you cannot move, copy, or delete it until the queue entry is resolved. Remove the item from the Queue to unlock.

❓
Unknown Types

Items that don't match any pattern show ❓ Unknown in the type dropdown and are greyed out. Use the dropdown to manually assign a type — this enables the checkbox and lets you queue the item. Select ⚙ Add Type at the bottom of the dropdown to add new media types in Settings → Library.

Filters

🔘
All / Files / Folders

Quickly narrow the list to only files or only folders. Click the active button again to clear the filter.

🏷️
Media Type Filters

The TV, Movie, Music, Games buttons (and any custom types you add) filter the list to only that detected type. Generated dynamically from your configured media types in Settings → Library.

❓
Toggle Unrecognized Files

The ❓ button in the filter bar shows or hides files whose type couldn't be detected. Active (lit) = unrecognized files are visible; inactive = they're hidden. The default is shown. You can also set the default in Settings → Appearance → File List.

Sorting

↕
Column Headers

Click any column header — Name, Source, Type, Size, or Modified — to sort by that column. Click again to reverse the order. An arrow indicator (↑ / ↓) shows the active sort.

📏
Folder Sizes

Folder sizes are computed in the background after loading. Sorting by Size uses the computed value once available, otherwise groups uncomputed folders together.

Move & Copy

📦
Move

Relocates the selected files or folders to the chosen destination. The original is removed after a successful transfer.

🗒️
Copy

Duplicates the selected files or folders to the destination. The original is kept in place.

Folder Options

📁
Place in Subfolder

Wraps the selected file(s) inside a new named subfolder at the destination — e.g. Movie.Name.2024.mkv → /Movies/Movie Name (2024)/Movie.Name.2024.mkv. Auto-checked for non-TV files by default (configurable in Behavior settings).

📤
Move Contents Only

When a folder is selected, moves its contents to the destination rather than the folder itself. Auto-checked for TV episode folders by default (configurable in Behavior settings).

📺
Season Folder

Auto-routes TV episodes into a Season XX subfolder at the destination — detected from the episode filename (e.g. S02E05). Auto-checked for TV by default (configurable in Behavior settings).

Rename

✏️
Rename Presets

Optionally rename files during the action. Presets include: Leave as is, Spaces → Periods, Spaces (Title Year), Periods (Title.Year), Strip extras, and Custom (type your own name). A live preview shows the result before queuing.

Delete

🗑
Move to Trash

Sends files to your configured Trash folder. Recoverable via the 🗑 Trash button in the header. Requires a Trash folder to be set in Settings → Housekeeping.

☠️
Permanent Delete

Irreversibly removes files. Cannot be undone. Use with caution.

Mixed Type Safety

⚠️
Block Mixed Queuing

When enabled in Behavior settings, Add to Queue and Run Now are disabled if different media types (e.g. TV + Movie) are selected together, or if files from multiple different TV shows are mixed. The ⚠ indicator shows what conflict is detected. Enable in Settings → Behavior.

Queue

➕
Add to Queue

Stages the configured action without running it. You can add as many actions as you need, selecting different files and destinations each time, then run them all at once.

▶️
Run Queue

Executes all queued actions in order. Progress is shown in the queue panel. Actions run sequentially — each waits for the previous to finish.

⚡
Run Now

Skips the queue and runs the selected action immediately. Useful when you only have one thing to do and don't need to batch.

✕
Removing Items

Click ✕ on a queue entry to remove it. Removing an item from a locked folder's queue entry will unlock the folder. Use 🗑 Clear to remove all queued items at once.

History

🕐
Action Log

Every completed action is logged in the History tab of the queue panel. Use the search box to filter by file name, source, or destination.

↩️
Revert

Move, Copy, Trash, and Restore actions can be reverted from the History tab. Permanent deletes cannot be undone.

📅
Retention

Configure how long history is kept in Settings → Housekeeping. Options range from 7 days to Forever.

Library

📂
Sources

Folders that DownloadsManarr scans and displays in the file list. Add your Downloads folder or any watched directory. Use Browse to pick a path.

🎯
Destinations

Target folders for each media type. Each type (TV, Movie, Music, Games…) gets its own section so files land in the right library automatically.

🏷️
Media Types

The set of media types shown as filter buttons and destination groups. Add custom types like Books, Podcasts, Anime, Courses. Remove types you don't use. Changes apply immediately across the whole app.

Behavior

📂
File Browser

Show Sources & Destinations by default — controls whether the collapsible Sources & Destinations bar is expanded when the page loads. Saved per user.

📋
Queue Defaults

Controls which action-bar checkboxes are pre-checked when you select files. Place in subfolder auto-checks for non-TV files; Move contents only auto-checks for TV episode folders; Delete original folder if empty removes the source folder after a move if nothing is left behind.

⚠️
Mixed Type Safety

Blocks "Add to Queue" and "Run Now" when different media types are selected together (e.g. TV + Movie), or when files from multiple TV shows are mixed.

Housekeeping

🗑
Trash Mode

Controls how "Move to Trash" works. Off — disables soft delete entirely. Auto (default) — creates a .Trash folder beside each source file at execute time; no pre-configuration required. Custom — send everything to a single explicit path you specify. The 🗑 Trash panel lists and lets you restore or permanently delete items from whichever mode is active.

📅
History Retention

How long completed action logs are kept before being automatically pruned. Options: 7 days, 30 days, 90 days, 1 year, or Forever.

Appearance

🌙
Theme

Choose from Dark, OLED (pure black for OLED displays), Light, or Custom — where you can set your own background, surface, and text colors. Theme is saved per user and persists across all browsers and devices.

🎨
Accent Color

Change the highlight color used for buttons, active states, and links. Choose from Purple, Blue, Green, Amber, or a fully custom hex color. Saved per user alongside your theme preferences.

🔡
Text Size

Adjust the base font size of the interface. Choose from Small (12px), Medium (14px, default), or Large (16px). Saved per user and applied instantly without a page reload.

❓
Show Unrecognized Files

When enabled (the default), files whose type couldn't be detected appear in the file list as ❓ Unknown. Uncheck to hide them. You can also toggle this on the fly with the ❓ button in the filter bar.

Authentication

🔐
Login

Set DL_MANAGARR_ADMIN_USERNAME and DL_MANAGARR_ADMIN_PASSWORD in your Docker environment to enable a login screen. Check Keep me logged in for a 30-day persistent session; leave it unchecked for a session cookie that expires when the browser closes (24h server-side cap). Omit both variables to run without authentication.

⏏
Logout

The Logout button appears in the header when authentication is enabled. Clicking it invalidates your session immediately and returns you to the login screen.
